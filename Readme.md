## Installation

Install dependencies.

```js
git clone https://github.com/localhostport80/simplecalc && cd simplecalc
npm install
```

## Building

### DevApp

Run it inside Ionic DevApp.

```bash
ionic serve --devapp
```

### Release

Build it for the specific platform.

```bash
ionic cordova build --release <platform>
